How to commit:
1. Navigate to your Netbeans projects folder then open command prompt.
2. run 'git push' to login to your github account.
3. run git clone https://github.com/cyyxeia/student-enrollment-system.git
4. make changes
5. run git add . (including the period)
6. run 'git status' to check your changes
7. run 'git commit -m "your commit message here"'
